# ✍️ كاتب المحتوى العربي

مهارة لكتابة محتوى عربي عالي الجودة: مقالات، نصوص تسويقية، توثيق تقني، منشورات.

## متى تُستخدم
- طلب كتابة مقال أو منشور أو وصف منتج بالعربية.
- صياغة توثيق تقني عربي واضح.
- تحرير وتحسين نص عربي موجود.

## المبادئ
1. **الوضوح أولاً** — جملة واحدة = فكرة واحدة. تجنّب الحشو.
2. **الفصحى المبسّطة** — لغة سليمة دون تعقيد ممل.
3. **بنية واضحة** — عنوان جذّاب، مقدمة، نقاط، خلاصة.
4. **نبرة مناسبة للجمهور** — رسمية للتقارير، ودّية للتسويق.
5. **علامات الترقيم العربية** — استخدم «الاقتباس» و: النقطتين بدقة.

## خطوات التنفيذ
1. حدّد: الجمهور + الهدف + الطول المطلوب.
2. اكتب عنواناً يثير الفضول أو يَعِد بقيمة.
3. مقدمة من سطرين تذكر المشكلة/الوعد.
4. الجسم: نقاط مرقّمة أو عناوين فرعية.
5. خلاصة + دعوة لإجراء (CTA) إن كان تسويقياً.
6. راجع: احذف الكلمات الزائدة، وحّد الضمائر والأزمنة.

## قالب سريع
```
# [عنوان جذّاب]
[مقدمة: المشكلة أو الوعد في سطرين]

## [العنصر 1]
[شرح موجز]

## [العنصر 2]
[شرح موجز]

## الخلاصة
[جملة ختامية + دعوة لإجراء]
```

## أخطاء يجب تجنّبها
- الترجمة الحرفية من الإنجليزية (يُفقد السلاسة).
- الجمل الطويلة المتشعّبة.
- الإفراط في المحسّنات اللفظية على حساب المعنى.
