# ⚡ محلّل فواتير الكهرباء

مهارة لتحليل فواتير الكهرباء: قراءة العدّاد، حساب الاستهلاك، تطبيق الشرائح،
وتقدير التكلفة. مرتبطة بمشروع `electricity-billing-pwa`.

## متى تُستخدم
- إدخال قراءتي عدّاد (سابقة/حالية) لحساب الاستهلاك.
- حساب الفاتورة وفق شرائح تعرفة متدرّجة.
- تحليل استهلاك شهري ومقارنته بالأشهر السابقة.

## المفاهيم الأساسية
- **الاستهلاك (kWh)** = القراءة الحالية − القراءة السابقة.
- **الشرائح المتدرّجة**: كل شريحة لها سعر مختلف؛ يُحسب كل جزء بسعره.
- **التكلفة الكلية** = Σ (كمية كل شريحة × سعرها) + رسوم ثابتة.

## خوارزمية حساب الشرائح
```python
def احسب_الفاتورة(الاستهلاك, الشرائح, رسوم_ثابتة=0.0):
    """
    الشرائح: قائمة (الحد_الأعلى, السعر). آخر شريحة حدّها الأعلى = اللانهاية.
    مثال: [(100, 5), (200, 8), (float('inf'), 12)]
    """
    متبقٍ = الاستهلاك
    تكلفة = 0.0
    سابق = 0
    for الحد, السعر in الشرائح:
        كمية = min(متبقٍ, الحد - سابق)
        if كمية <= 0:
            break
        تكلفة += كمية * السعر
        متبقٍ -= كمية
        سابق = الحد
    return round(تكلفة + رسوم_ثابتة, 2)
```

## خطوات التنفيذ
1. تحقّق: القراءة الحالية ≥ السابقة (وإلا تنبيه خطأ إدخال).
2. احسب الاستهلاك.
3. طبّق الشرائح بالخوارزمية أعلاه.
4. أضف الرسوم الثابتة والضرائب إن وُجدت.
5. اعرض تفصيلاً: الاستهلاك + تكلفة كل شريحة + الإجمالي.
6. (اختياري) قارن بالشهر السابق وأظهر نسبة التغيّر.

## نصائح
- تعامل مع العملة المحلية (الريال اليمني) بدقة منزلتين.
- نبّه عند الاستهلاك الشاذ (قفزة > 50% عن المتوسط).
