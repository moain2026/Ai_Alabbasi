## 🛡️ 2. مبادئ التخفيف العامة

1. **اختبر قبل وبعد كل تعديل** — `python3 run.py`, `brain.py`, `tools.py`, `knowledge.py`.
2. **أضِف ثم احذف** — لا تحذف الأصلي قبل إثبات عمل البديل.
3. **commit ذرّي** — كل مهمة = commit واحد → revert آمن عند الخطأ.
4. **كل LLM عبر brain.think()** — لا استثناءات (حماية Swappable Brain).
5. **الأمان داخل sandbox** — `_safe_path` يحرس كل عمليات الملفات.
