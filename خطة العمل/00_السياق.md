## 🔑 5. المفاهيم التقنية الأساسية (المعجم)

| المفهوم | الشرح المختصر |
|---------|---------------|
| **Swappable Brain** | تبديل مزوّد LLM بسطر واحد في YAML (8 أدمغة). أي تعديل يجب أن يعمل مع الـ8 جميعاً. |
| **ReAct loop** | Reason + Act: فكرة → أداة → ملاحظة → تكرار (مطبّق في `agent.py`). |
| **CodeAct** | تنفيذ كود Python كآلية فعل موحّدة بدل function calling الثابت (نمط Manus). |
| **Event Stream** | سجل زمني مركزي لكل: Message / Action / Observation / Plan / Knowledge. |
| **todo.md Recitation** | تحديث ملف todo باستمرار وحقنه في نهاية السياق لمكافحة "الضياع في المنتصف". |
| **Hybrid Search** | FTS5 (كلمات) + Vector embeddings (معاني) مدموجين بـ RRF. |
| **RRF** | Reciprocal Rank Fusion — دمج نتائج بحثين بترتيب موحّد. |
| **Progressive Disclosure** | كشف تدريجي 3 طبقات للمهارة: اكتشاف (~80 token) → تفعيل (~2000 token) → تنفيذ (عند الطلب). |
| **SKILL.md** | معيار مفتوح (Anthropic، ديسمبر 2025): YAML frontmatter + Markdown body. |
| **_safe_path** | حارس أمني يمنع الكتابة خارج `projects/`. |

## ⚖️ 7. القيود الذهبية (Golden Rules) — لا تُكسر أبداً

1. **احترم Swappable Brain** — كل تعديل يعمل مع الأدمغة الـ8.
2. **احترم `_safe_path`** — كل عمليات الملفات داخل `projects/` فقط.
3. **لا تبعيات ثقيلة بلا تبرير** — PyYAML فقط + اختيارية وراء ADR (sentence-transformers / daytona-sdk / pytest).
4. **اللغة العربية** — docstrings/comments/logs عربية، أسماء المتغيرات إنجليزية، commits عربي + emoji + english refs.
5. **Atomic Commits** — كل commit = مهمة واحدة (يسمح بـ revert آمن).
6. **بحث web قبل كل مهمة** — لا تعتمد على المعرفة فقط، دوّن البحث في PLAN.
7. **لا تحذف الأصلي** — أضف أولاً، اختبر، ثم احذف (مع ذكر السبب).
8. **شغّل tests بعد كل تعديل** — 100% نجاح قبل أي push.
9. **لا تلمس main مباشرة** — كل العمل على `upgrade/v2-manus-patterns`.
10. **السرية أولاً** — لا ترفع `config/.env`، لا مفاتيح في الكود.
